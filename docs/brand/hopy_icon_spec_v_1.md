# HOPY Icon Spec v1 / HOPY アイコン スペック v1

## 1. Purpose / 目的

This icon defines HOPY as a calm companion that helps people move from confusion toward decision.

HOPYを、混線から決定へ向かう流れに静かに寄り添う存在として定義するアイコンです。

The icon must feel quiet, refined, and long-lasting.
It should not look like a navigation app, an adventure tool, or a flashy productivity logo.

見た目は、静かで、上品で、長く使えることを前提にします。
ナビアプリ、冒険ツール、派手な生産性アプリのように見えてはいけません。

## 2. Core Meaning / 中心意味

HOPY’s icon is based on the flow:

**Confusion → Exploration → Organization → Convergence → Decision**

HOPYのアイコンは、次の流れを土台にします。

**混線 → 模索 → 整理 → 収束 → 決定**

The icon expresses this through:

- a compass motif
- five circles representing the five thought states
- an upward needle that symbolizes dignity, rising, and moving toward decision

この流れを、次の要素で表現します。

- コンパスのモチーフ
- 思考の5段階を表す5つの○
- 凛と上を指し、上昇と決定への進行を示す針

## 3. Official Concept Statement / 正式コンセプト文

**Place confusion at the center, open the flow of thought toward four directions, and let the needle stand upright toward the northern point of decision.**

**混線を中心に置き、思考の流れを4方向へ開きながら、針は凛と北の決定へ向かう。**

## 4. Structural Definition / 構造定義

### 4.1 Base Structure / 基本構造

The icon is composed of:

- one central circle
- four outer circles
- one compass needle
- one quiet circular frame or implied compass field

アイコンは次の要素で構成します。

- 中央の○ 1つ
- 外周の○ 4つ
- コンパスの針 1本
- 静かな円形フレーム、またはそれを感じさせるコンパス面

### 4.2 Meaning of Each Circle / 各○の意味

- **Center circle**: Confusion
- **North circle**: Decision
- **East circle**: One intermediate state
- **West circle**: One intermediate state
- **South circle**: One intermediate state

- **中心の○**: 混線
- **北の○**: 決定
- **東の○**: 中間状態のいずれか1つ
- **西の○**: 中間状態のいずれか1つ
- **南の○**: 中間状態のいずれか1つ

The three remaining stages — Exploration, Organization, and Convergence — are placed across East, West, and South.
The exact placement can be adjusted visually, but North must always represent Decision, and the Center must always represent Confusion.

残る3状態である 模索・整理・収束 は、東・西・南へ配置します。
細かな配置順は見た目の調整で決めてよいですが、北＝決定、中心＝混線 は固定です。

## 5. Directional Meaning / 方位の意味

### 5.1 North / 北

North is the most important outer point.
It represents:

- Decision
- rising upward
- dignity
- clarity
- standing tall

北は、外周4点の中で最も重要な点です。
意味は次です。

- 決定
- 上昇
- 凛とすること
- 明瞭さ
- 背筋が伸びる感覚

### 5.2 Needle / 針

The compass needle must point upward as the default official expression.

コンパスの針は、正式表現として上向きを基本にします。

Meaning:

- choosing direction
- moving out of confusion
- aligning oneself
- becoming calm and clear
- heading toward Decision

意味は次です。

- 向かう方向を定めること
- 混線から抜け出すこと
- 自分を整えること
- 静かに明瞭になること
- 決定へ向かうこと

The needle should not look aggressive or adventurous.
It should feel calm, precise, and quietly strong.

針は攻撃的にも冒険的にも見えてはいけません。
静かで、正確で、強さを押しつけない見え方にします。

## 6. Visual Tone / 見た目の空気感

The icon must feel:

- calm
- refined
- elegant
- minimal
- trustworthy
- long-lasting

アイコンが持つべき空気感は次です。

- 静か
- 上品
- 洗練
- ミニマル
- 信頼できる
- 長く使える

The icon must avoid:

- excessive realism
- strong metallic effects
- loud gradients
- aggressive contrast
- game-like or map-like styling
- cheap “app icon” effects

避けるべきものは次です。

- 過剰な写実性
- 強い金属感
- うるさいグラデーション
- 強すぎるコントラスト
- ゲームや地図アプリのような見え方
- 安っぽいアプリアイコン風の演出

## 7. Circle Behavior / ○の扱い

The five circles are not decorative dots.
They represent HOPY’s five thought states.

5つの○は装飾ではありません。
HOPYの思考5段階そのものを表します。

They should feel like:

- quiet state points
- a subtle emotional/cognitive map
- part of one calm system

○は次のように感じられるべきです。

- 静かな状態点
- 感情と思考の流れを示す繊細な地図
- ひとつの落ち着いた体系の一部

They should not dominate the icon more than the compass itself.
They are a signature layer, not the entire icon.

ただし、コンパス本体より主役になりすぎてはいけません。
○は署名的な層であり、アイコン全体の主役そのものではありません。

## 8. Relative Hierarchy / 優先順位

Visual priority should be:

1. Needle direction
2. Overall compass form
3. North / Decision point
4. Center / Confusion point
5. Remaining three state points

見た目の優先順位は次です。

1. 針の向き
2. 全体としてのコンパス形
3. 北の決定点
4. 中心の混線点
5. 残り3つの状態点

## 9. Color Role Definition / 色の役割定義

### 9.1 Official Principle / 基本原則

Color should communicate meaning quietly, not loudly.

色は、意味を静かに伝えるために使います。
主張しすぎてはいけません。

### 9.2 Required Role Rules / 必須ルール

- **Center circle** must represent Confusion
- **North circle** must use the official Decision color
- **Other three outer circles** should support the remaining states with quieter tones
- The needle should harmonize with the system and remain readable at small size

- **中心の○** は混線を表すこと
- **北の○** は正式な決定色を使うこと
- **他の3つの外周○** は残り状態を静かな色で支えること
- **針** は全体と調和しつつ、小さくしても読めること

### 9.3 Tone Rule / トーンルール

Colors should stay within a calm, premium, restrained family.
No playful rainbow treatment.
No loud saturation battle.

色は、静かで上質で抑制のある範囲に収めます。
虹のような遊び方や、彩度の強いぶつかり合いは避けます。

## 10. Shape Rule / 形のルール

The icon should still be recognizable when very small.

アイコンは、小さくしても認識できなければなりません。

Therefore:

- circles must remain legible at favicon scale
- the needle must remain clear at small size
- fine decorative lines should be minimized
- the design should work in monochrome or reduced-color contexts

そのため、次を守ります。

- faviconサイズでも○が読めること
- 小サイズでも針が明確であること
- 細かすぎる装飾線は減らすこと
- 単色や減色でも成立すること

## 11. Relationship to Hero Compass / ヒーローCompassとの関係

The homepage hero compass and the icon are connected, but they are not the same object.

トップページのヒーローCompassとアイコンは、つながっていますが同一物ではありません。

- **Hero compass** = immersive brand experience
- **Icon** = compact brand symbol

- **ヒーローCompass** = 没入感のあるブランド体験
- **アイコン** = 小さく凝縮されたブランド記号

The icon should inherit the meaning of the hero, not literally miniaturize the hero.

アイコンは、ヒーローをそのまま縮小するのではなく、その意味を受け継ぐものとします。

## 12. Official Brand Interpretation / 正式なブランド解釈

This icon represents:

- starting from confusion
- sensing possible directions
- being guided without pressure
- rising toward clarity
- standing upright toward decision

このアイコンが表すものは次です。

- 混線から始まること
- 進みうる方向を感じること
- 強制されずに導かれること
- 明瞭さへ向かって上がっていくこと
- 決定へ向かって凛と立つこと

It should feel like a quiet companion, not a commanding system.

命令するシステムではなく、静かに寄り添う伴走者として感じられるべきです。

## 13. Usage Principle / 使用原則

This single icon system should become the parent source for:

- favicon
- PWA icon set
- apple-touch-icon
- app launcher icon
- lightweight brand mark usage where appropriate

この単一のアイコン体系を、次の親ソースにします。

- favicon
- PWAアイコン一式
- apple-touch-icon
- アプリランチャーアイコン
- 必要に応じた軽量ブランドマーク

## 14. Production Principle / 制作原則

The master source should be created once and then exported into all required outputs.

元データは1回だけ正しく作り、その後に必要な出力へ展開します。

Recommended output flow:

1. master SVG
2. favicon assets
3. PWA PNG set
4. apple-touch-icon
5. any additional brand derivatives

推奨する出力順は次です。

1. 元SVG
2. favicon用アセット
3. PWA用PNG一式
4. apple-touch-icon
5. 必要な派生ブランド素材

## 15. Final Judgment Criteria / 最終判断基準

The icon is correct only if all of the following feel true:

- it feels like HOPY
- it feels calm
- it feels refined
- it still works when tiny
- it suggests movement toward decision
- it does not feel like a generic compass app
- it quietly contains the five-state philosophy

次のすべてを満たすとき、このアイコンは正しいと判断します。

- HOPYらしく感じる
- 静かに感じる
- 上品に感じる
- 小さくしても成立する
- 決定へ向かう流れを感じる
- ありふれたコンパスアプリに見えない
- 5段階思想が静かに内包されている

## 16. Official Direction Summary / 正式方向まとめ

**HOPY icon = Compass × five thought-state circles**

**HOPYアイコン = コンパス × 思考5段階の○**

Center = Confusion.
North = Decision.
Needle = dignified upward direction.
The whole symbol quietly expresses movement from confusion toward decision.

中心 = 混線。
北 = 決定。
針 = 凛と上を向く方向。
全体として、混線から決定へ向かう静かな移行を表現します。
