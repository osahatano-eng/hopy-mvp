\# HOPY Icon SVG Structure v1 / HOPY アイコンSVG構造 v1



\## 1. Purpose / 目的



This document defines the official structural rules for the HOPY master SVG icon.



この文書は、HOPYの元SVGアイコンにおける正式な構造ルールを定義するものです。



The goal is to create one stable master structure that can later be exported into favicon, PWA icons, apple-touch-icon, and other brand outputs.



目的は、favicon、PWAアイコン、apple-touch-icon、その他のブランド出力へ展開できる、安定した単一の元構造を作ることです。



\---



\## 2. Core Principle / 中心原則



The HOPY icon is not a generic compass mark.



HOPYアイコンは、汎用的なコンパスマークではありません。



It is a symbol of moving from confusion toward decision.



混線から決定へ向かう流れを象徴する記号です。



Its structure must quietly express:



構造として、次を静かに表現する必要があります。



\- confusion at the center / 中心に混線があること

\- four directional state points around it / 周囲に4方向の状態点があること

\- a needle that rises upward / 針が上へ向かうこと

\- movement toward the northern point of decision / 北の決定へ向かうこと



\---



\## 3. Official Structure Summary / 正式構造まとめ



\*\*HOPY icon structure = one center point + four outer points + one upward needle + one calm compass field\*\*



\*\*HOPYアイコン構造 = 中心点1つ + 外周点4つ + 上向きの針1本 + 静かなコンパス面\*\*



\---



\## 4. Structural Elements / 構成要素



\### 4.1 Center Point / 中心点



The center point represents Confusion.



中心点は混線を表します。



Rules:



\- exactly one circle / 円は1つ

\- always placed at the exact center / 必ず正確な中心に置く

\- visually smaller than the total icon, but not too small to disappear / 全体に対しては小さめだが、消えるほど小さくしない

\- must remain visible at favicon scale / faviconサイズでも見えること



\### 4.2 Outer Four Points / 外周4点



The outer four points represent the directional thought states surrounding the center.



外周4点は、中心を取り囲む方向的な思考状態を表します。



Rules:



\- one point at North / 北に1点

\- one point at East / 東に1点

\- one point at South / 南に1点

\- one point at West / 西に1点

\- these four points must form a stable cross-like balance / 4点は十字に近い安定したバランスを作ること

\- North must always represent Decision / 北は必ず決定を表すこと



\### 4.3 Needle / 針



The needle represents direction, intention, posture, and movement toward clarity.



針は、方向、意志、姿勢、明瞭さへの移動を表します。



Rules:



\- default official direction is upward / 正式な基本方向は上向き

\- must visually connect the center with the upward direction / 視覚的に中心と上方向をつなぐこと

\- should feel calm and precise / 静かで正確に感じられること

\- should not look aggressive, sharp in a violent way, or adventurous / 攻撃的、暴力的に鋭い、冒険的に見えてはいけない



\### 4.4 Compass Field / コンパス面



The icon may include a circular field or a softly implied compass area.



アイコンは、円形の面、またはやわらかく示唆されたコンパス領域を含んでよいです。



Rules:



\- the field must support the symbol, not dominate it / 面は記号を支えるが、主役にならないこと

\- outer ring may be used if needed / 必要なら外周リングを使ってよい

\- decorative detail must be restrained / 装飾は抑制すること



\---



\## 5. State Placement Rules / 状態配置ルール



\### 5.1 Fixed Meaning / 固定意味



The following meanings are fixed:



次の意味は固定です。



\- Center = Confusion / 中心 = 混線

\- North = Decision / 北 = 決定



\### 5.2 Adjustable Meaning / 調整可能な配置



The remaining three states may be arranged across East, South, and West:



残る3状態は東・南・西へ配置します。



\- Exploration / 模索

\- Organization / 整理

\- Convergence / 収束



Their exact visual arrangement can be adjusted for beauty and balance, but the meaning of each color must remain consistent.



見た目の美しさとバランスのために細かな配置順は調整してよいですが、各色の意味は固定します。



\---



\## 6. Geometric Balance / 幾何バランス



The icon should feel centered, upright, and composed.



アイコン全体は、中心が定まり、上に向かい、整って感じられる必要があります。



Rules:



\- the center point is the geometric anchor / 中心点を幾何学的な基準点にする

\- North / East / South / West should keep balanced spacing / 東西南北はバランスの取れた間隔にする

\- the upward needle must visually feel slightly more important than the other directions / 上向きの針は他方向よりわずかに重要に感じられること

\- North may be subtly emphasized / 北はわずかに強調してよい



\---



\## 7. Relative Size Rules / 相対サイズルール



These are proportional rules, not final pixel values.



これは最終ピクセル値ではなく、比率ルールです。



\### 7.1 Center Circle / 中心円



\- smaller than any outer point emphasis area / 外周点の強調領域より小さめ

\- large enough to remain identifiable / 識別できる大きさを保つ



\### 7.2 Outer State Circles / 外周状態円



\- visually similar size family / 基本は同系サイズ

\- North may be slightly more prominent / 北のみわずかに目立たせてよい

\- none of the outer circles should overpower the needle / 外周円が針を食わないこと



\### 7.3 Needle / 針



\- longer than the radius distance from center to outer points / 中心から外周点までの距離に対して十分な長さを持つこと

\- thin enough to stay elegant / 上品さを保てる細さ

\- thick enough to survive small-size export / 小サイズ出力でも消えない太さ



\### 7.4 Ring / リング



\- thinner than the needle / 針より細いこと

\- quieter than the state circles / 状態円より静かな見え方にすること



\---



\## 8. Small-Size Simplification Rule / 小サイズ簡略ルール



At very small sizes, the icon must remain recognizable.



極小サイズでも、アイコンとして認識できなければなりません。



Therefore:



\- keep only the essential shapes / 必須形状だけを残す

\- avoid tiny decorative marks / 細かい装飾は入れない

\- preserve center point, north point, and upward needle at minimum / 最低でも中心点・北点・上向き針は守る

\- if needed, reduce ring and secondary details first / 必要ならリングや補助要素から先に削る



\---



\## 9. Favicon Rule / faviconルール



For favicon use, readability is more important than symbolic richness.



faviconでは、象徴の豊かさよりも視認性を優先します。



Rules:



\- five-state philosophy must remain present, but in simplified form / 5段階思想は簡略形でも残す

\- center and north must stay distinguishable / 中心と北は識別可能であること

\- needle must remain clear / 針は明確であること

\- avoid over-detailed ring structures / リングは細かくしすぎないこと



\---



\## 10. PWA Icon Rule / PWAアイコンルール



PWA icons are viewed larger than favicons, so they can preserve more structure.



PWAアイコンはfaviconより大きく見られるため、より多くの構造を保持できます。



Rules:



\- all five points should be preserved / 5点は保持する

\- outer ring can remain if it helps coherence / 全体のまとまりに効くなら外周リングを残してよい

\- the icon should remain centered within safe export margins / 書き出し時の安全余白内にきれいに収める

\- maskable export should protect the center and north point / maskable書き出しでは中心と北点を守ること



\---



\## 11. Relationship to Hero Compass / ヒーローCompassとの関係



The hero compass and the icon share meaning, but not literal detail.



ヒーローCompassとアイコンは意味を共有しますが、細部まで同一ではありません。



The master SVG should inherit:



元SVGは、次を受け継ぐべきです。



\- calm presence / 静かな存在感

\- upward dignity / 上へ向かう凛とした感じ

\- thoughtful direction / 思考を持った方向性

\- emotional quietness / 感情の静けさ



The master SVG should not copy the hero 1:1.



元SVGは、ヒーローをそのまま1:1で写してはいけません。



\---



\## 12. Export Parent Rule / 書き出し親ルール



This SVG is the parent source.



このSVGは親ソースです。



All derivatives must come from this structure:



すべての派生物は、この構造から作ります。



\- favicon

\- PWA icons

\- apple-touch-icon

\- app launcher icon

\- lightweight brand mark uses



No derived asset should invent a different structure.



派生アセット側で別構造を勝手に作ってはいけません。



\---



\## 13. Final Judgment Criteria / 最終判断基準



The SVG structure is correct only if all of the following are true:



次のすべてを満たすとき、このSVG構造は正しいと判断します。



\- it feels like HOPY / HOPYらしい

\- it expresses movement from confusion to decision / 混線から決定への流れを表している

\- the needle feels upright and calm / 針が上向きで静かに感じられる

\- the center feels like the origin / 中心が出発点に感じられる

\- the north point feels special / 北点が特別に感じられる

\- the whole structure remains elegant / 全体が上品である

\- it survives favicon scale / faviconサイズでも成立する

\- it does not become a generic compass app icon / 汎用コンパスアプリに見えない



\---



\## 14. Official Summary / 正式まとめ



\*\*HOPY master SVG structure v1\*\*



\- Center point = Confusion / 混線

\- North point = Decision / 決定

\- East / South / West = Exploration, Organization, Convergence / 模索・整理・収束

\- Needle = upward dignity and direction / 上向きの凛とした方向

\- Compass field = calm structural support / 静かな構造的支え



The whole icon quietly expresses movement from confusion toward decision.



アイコン全体で、混線から決定へ向かう静かな移行を表現します。
