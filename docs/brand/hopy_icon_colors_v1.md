\# HOPY Icon Colors v1 / HOPY アイコンカラー v1



\## 1. Purpose / 目的



This document defines the official color roles for the HOPY icon system.



この文書は、HOPYアイコン体系における正式な色の役割を定義するものです。



The goal is not to make the icon flashy.

The goal is to make the icon feel calm, refined, meaningful, and long-lasting.



目的は、派手なアイコンを作ることではありません。  

静かで、上品で、意味があり、長く使えるアイコンにすることです。



\---



\## 2. Core Principle / 中心原則



Color must support meaning quietly.



色は、意味を静かに支えるために使います。



The HOPY icon should never feel noisy, playful, childish, or overly app-like.



HOPYのアイコンは、うるさく、子どもっぽく、軽すぎて、よくあるアプリっぽく見えてはいけません。



The color system must feel:



色の体系は次の空気感を持つ必要があります。



\- calm / 静か

\- refined / 上品

\- thoughtful / 思慮深い

\- trustworthy / 信頼できる

\- long-lasting / 長く使える



\---



\## 3. Five State Colors / 5状態の色



\### 3.1 Confusion / 混線



\*\*Role / 役割\*\*  

The center point.  

The starting state.  

A state that is not yet sorted, but still has depth and possibility.



中心点。  

出発点。  

まだ整理されていないが、深さと可能性を持つ状態。



\*\*Official Color / 正式色\*\*  

`#6B7280`



\*\*Meaning / 意味\*\*  

\- uncertainty / まだ定まっていない

\- fog / 霧

\- inward complexity / 内側の複雑さ

\- the place before direction / 方向が決まる前



\---



\### 3.2 Exploration / 模索



\*\*Role / 役割\*\*  

The first outward search.  

The movement of looking for possible directions.



最初の外向きの探り。  

進みうる方向を探している動き。



\*\*Official Color / 正式色\*\*  

`#3B82F6`



\*\*Meaning / 意味\*\*  

\- search / 探す

\- openness / 開く

\- possibility / 可能性

\- air and movement / 風と動き



\---



\### 3.3 Organization / 整理



\*\*Role / 役割\*\*  

The state where scattered elements begin to settle into order.



散らばった要素が、少しずつ整っていく状態。



\*\*Official Color / 正式色\*\*  

`#14B8A6`



\*\*Meaning / 意味\*\*  

\- balance / バランス

\- arrangement / 整える

\- calm structure / 落ち着いた構造

\- breathing room / 呼吸ができる感じ



\---



\### 3.4 Convergence / 収束



\*\*Role / 役割\*\*  

The state where meaning gathers and the direction becomes narrower and clearer.



意味が集まり、方向が狭まり、明確さが増していく状態。



\*\*Official Color / 正式色\*\*  

`#8B5CF6`



\*\*Meaning / 意味\*\*  

\- gathering / 集まる

\- depth / 深まり

\- focus / 焦点

\- nearing resolution / 解像度が上がる手前



\---



\### 3.5 Decision / 決定



\*\*Role / 役割\*\*  

The northern point.  

The final state.  

The state of standing upright and moving with clarity.



北の点。  

最終状態。  

凛として、明瞭に進む状態。



\*\*Official Color / 正式色\*\*  

`#F59E0B`



\*\*Meaning / 意味\*\*  

\- decision / 決定

\- clarity / 明瞭さ

\- rise / 上昇

\- dignity / 凛とすること

\- a steady forward will / 静かで強い意志

針は、方向、姿勢、意志を表します。

正式表現では上向きを基本とし、静かで、正確で、凛として見える必要があります。



\*\*Official Color / 正式色\*\*  

`#1F2937`



\*\*Meaning / 意味\*\*  

\- precision / 正確さ

\- inner axis / 内なる軸

\- calm strength / 静かな強さ

\- guidance without pressure / 圧をかけない導き

\## 6. Official Placement Meaning / 配置と色の意味



The color system follows this structure:



色の配置は次の構造に従います。



\- \*\*Center\*\* = Confusion / 混線

\- \*\*North\*\* = Decision / 決定

\- \*\*East / West / South\*\* = Exploration, Organization, Convergence / 模索・整理・収束

\- \*\*Needle\*\* = inner axis pointing upward / 上を指す内なる軸



North must always use the Decision color.



北は必ず決定色を使います。



The center must always use the Confusion color.



中心は必ず混線色を使います。



The remaining three states may be arranged visually among East, West, and South, but their colors must remain tied to their meanings.



残る3状態は東・西・南に見た目上の調整をしてよいですが、色の意味自体は固定します。



\---



\## 7. Tone Balance Rule / トーンの整え方



These five colors must not fight each other.



5色は互いにぶつかり合ってはいけません。



They should feel like one quiet family.



1つの静かな家族のように感じられるべきです。



Rules:



\- no neon treatment / ネオン化しない

\- no excessive saturation / 彩度を上げすぎない

\- no rainbow-like cheerfulness / 虹のような楽しさにしない

\- no harsh contrast competition / 強い対立にしない



The icon must still feel restrained and elegant even when all five state colors are visible.



5色すべてが見えても、全体は抑制的で上品でなければなりません。



\---



\## 8. Small Size Rule / 小サイズ時のルール



At favicon or app-icon scale, color must remain readable.



faviconやアプリアイコンのサイズでも、色は読める必要があります。



Therefore:



\- the Decision point must remain visually identifiable / 決定点は小さくても識別できること

\- the center Confusion point must not disappear / 中心の混線点は消えないこと

\- the needle must stay darker than the surrounding field / 針は周囲より暗く保つこと

\- rings and guide lines must remain secondary / リングや補助線は脇役のままであること



\---



\## 9. Light and Dark Background Usage / 明背景・暗背景での使い方



\### 9.1 Light Background / 明背景



Use the official colors as-is, with restrained contrast.



明背景では、正式色を基本そのまま使い、コントラストを抑制気味に保ちます。



\### 9.2 Dark Background / 暗背景



When used on dark backgrounds:



暗背景で使う場合は次を守ります。



\- base field may become slightly brighter / ベース面はやや明るくしてよい

\- ring lines may become softer and lighter / リング線はやわらかく明るくしてよい

\- state colors must remain recognizable / 状態色は識別性を保つこと

\- the icon must not become flashy / 派手になってはいけない



Dark mode should feel elegant, not glowing.



ダークモードでも、発光感ではなく上品さを優先します。

\## 12. Final Judgment Criteria / 最終判断基準



The color system is correct only if all of the following are true:



次のすべてを満たすとき、この色体系は正しいと判断します。



\- it feels like HOPY / HOPYらしい

\- it feels calm / 静か

\- it feels refined / 上品

\- it supports the five-state philosophy / 5段階思想を支えている

\- the north Decision point feels special / 北の決定点が特別に感じられる

\- the center Confusion point feels like the origin / 中心の混線点が出発点に見える

\- the icon works at small size / 小さくしても成立する

\- it does not feel like a generic app palette / よくあるアプリ配色に見えない
